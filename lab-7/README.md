# ITC505 Lab 7 — ExpressJS Mad Lib

This project implements a simple ExpressJS web app that serves a Mad Lib form and generates a story based on user input.

## Run Locally

```bash
npm install
node server.js local
```

Then open your browser and visit:

- http://localhost:8080/index.html
- http://localhost:8080/do_a_random

## Features
- ExpressJS static and POST routes
- `/do_a_random` random number generator
- Form validation and string interpolation
- Addendum included in the page footer
- Last Modified footer script
- Clean, responsive design
