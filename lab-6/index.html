    Addendum

    Problem solved: Implemented Bubble Sort on a user-provided list of integers, with options to sort ascending/descending and to display algorithm steps.

    How it was solved: The UI captures a comma-separated string, parses it into an array of integers, and passes it to a reusable bubbleSort function located in script.js. Event listeners on buttons trigger sorting or step-by-step tracing, updating the DOM with results and step states.

    Difficulties & solutions: Parsing invalid input (e.g., extra spaces or empty items) was handled by trimming, filtering blanks, and validating that all tokens are finite numbers. To avoid blocking the UI on large inputs when tracing steps, we limit displayed states and summarize when step counts are large.
