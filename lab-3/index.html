Lab 3: Serving Fonts (Google Fonts Method)

What I did (in my own words)

I had not to install any font files on my computer. Rather, I used Google font to serve the font. I also included a link tag in the <head> section of my HTML, which refers to the stylesheet of Google. This stylesheet is used to specify where the font file can be downloaded on the servers of Google. Later in my CSS, I included font-family: 'Creepster' in order to style the headings and big text. The browser will retrieve the font, it will be saved in the cache and this page will be rendered with the custom font.
