Lab 3: Serving Fonts (Google Fonts Method)

What I did (in my own words)

Instead of hosting font files myself, I used Google Fonts to serve the font. I added a <link> tag in the <head> of my HTML that points to Google’s stylesheet. This stylesheet tells the browser where to download the font file from Google’s servers. Then, in my CSS, I used font-family: 'Creepster' to apply it to headings and large text. The browser fetches the font, caches it, and displays this page using the custom font.
